rs-streetdice sound placeholders

These .ogg files are intentionally empty placeholders so the filenames are ready for buyers/server owners.
Replace the hype files with your own final .ogg voice lines and the dice_* files with short dice throw/land/pickup effects, then set Config.Sounds.enabled = true in config.lua.

Do not enable Config.Sounds until the placeholder files are replaced with valid OGG audio.
